# 


根據圖片中的內容，以下是整理後的命令：

1. 列出本機設定好的套件來源

```sh
dotnet nuget list source
```

2. 自訂專案的 NuGet 套件來源

```sh
dotnet new nuget.config
```

3. 加入自訂的 NuGet 套件來源路徑

```sh
dotnet nuget add source -n LocalNuGet G:\NuGet
```

4. 移除 NuGet 套件來源
```sh
dotnet nuget remove source "Package source"
```
